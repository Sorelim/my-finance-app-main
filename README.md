Клонируйте репозиторий проекта, используя следующую команду:

git clone https://github.com/Sorelim/my-finance-app-main
Перейдите в корневой каталог проекта:
cd personal-finance-app/frontend/

Установите зависимости проекта, выполнив команду:
npm install

Разработка
Чтобы запустить сервер разработки и предварительно просмотреть приложение, используйте следующую команду:
npm run dev

Тестирование
Запустите тесты, выполнив следующую команду:
npm run test
